# Luck

Luck is for players who want better random outcomes: more useful shard drops, stronger Common Lootbox openings, extra lootbox chances, and lucky bonuses or discounts.

## Live Status

| Trait | Status | What it changes |
| --- | --- | --- |
| Cards | Live | Turns in Common Lootbox openings. |
| Shards | Live | Chance that shard rolls prefer missing puzzle pieces. |
| Boxes | Live | Chance to receive an extra higher-rarity lootbox when starting a Common Lootbox opening. |
| Chance | Live | Probability that a lucky effect activates. |
| Bonus | Live | Strength of lucky rewards and lucky discounts. |

## Why Invest In Luck

Luck is not one single reward. It improves several random systems:

* Common Lootbox openings can last more turns.
* Chest Shard rolls can become more likely to fill missing pieces.
* Common Lootbox openings can award an extra higher-rarity lootbox.
* Some actions can receive lucky bonus rewards.
* Some costs or fees can receive lucky discounts.

Luck is strongest for players who open lootboxes, chase shard completion, craft often, sell items, or buy XP.

## Cards

Cards increases turns in Common Lootbox openings.

Common Lootboxes use a card flow. Each turn gives another chance to reveal a useful card before the opening ends. More turns means more chances to receive items from the opening.

Player impact:

* Better Common Lootbox openings.
* More chances to reveal item cards.
* More value from Common Lootboxes earned through Gold or Chest Shards.

Higher-rarity lootboxes do not use this card flow. They open directly and create one item at their rarity floor or higher.

## Shards

Shards helps Chest Shard rolls prefer missing puzzle pieces.

Mining Loot creates shard rolls. Luck Shards then influences which piece type is selected. If a player is missing one piece and has many duplicates, Shards makes the missing piece more likely, but it does not make duplicates impossible.

Player impact:

* Weekly shard sets finish more reliably.
* Free Common Lootbox opportunities become easier to complete.
* Works best with Mining Loot, because Mining creates the shard rolls that Luck can improve.

## Boxes

Boxes adds a chance to receive one extra higher-rarity lootbox when a Common Lootbox opening starts.

This is separate from the item cards inside the Common Lootbox opening. First the system checks whether the bonus box appears. If it appears, the bonus box rarity is rolled separately, with each next rarity much rarer than the previous one.

Player impact:

* Common Lootboxes can occasionally create an extra lootbox.
* The bonus box can be Uncommon or higher.
* This gives Luck a chance to turn common openings into higher-rarity opportunities.

## Chance

Chance controls whether a lucky effect activates on supported actions.

It does not decide how strong the result is. It only answers: did Luck happen this time?

Current lucky-effect areas include:

| Action area | Lucky effect |
| --- | --- |
| XP purchase | Bonus XP. |
| Crafting scrap | Bonus Essence. |
| Crafting level-up | Essence discount. |
| Marketplace sale | Fee discount. |

## Bonus

Bonus controls how strong a lucky result is after Chance activates.

There are two kinds of lucky effect:

| Type | Meaning |
| --- | --- |
| Reward | Adds extra value, such as bonus XP or bonus Essence. |
| Discount | Reduces a cost or fee, such as a level-up cost or marketplace sale fee. |

Player impact:

* Chance makes lucky events happen more often.
* Bonus makes those lucky events matter more.
* A strong Luck build usually wants both, not only one.
