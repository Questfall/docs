# Mining

Mining is for players who want completed quests to produce more reward value and more Chest Shard opportunities.

## Live Status

| Trait | Status | What it changes |
| --- | --- | --- |
| Power | Live | Mining Points earned from Quest Bounty. |
| Flow | Live | Bonus Mining Points while Flow is active. |
| Focus | Live | How long Flow stays active after a successful mining action. |
| Priority | Live | Position in moderation queues for completed quests. |
| Loot | Live | Chest Shard roll rate from completed quests. |

Mining does not replace quest quality or moderation. It changes the RPG reward layer after a quest completion is accepted or enters a queue.

## Why Invest In Mining

Mining is the direct questing attribute. It helps players:

* earn more Mining Points from the same accepted quest value;
* keep a Flow streak active between successful completions;
* get more Chest Shard rolls;
* finish weekly shard sets faster;
* move moderation-required completions higher in the queue.

A player who completes quests regularly gets the most direct value from Mining.

## Power

Power increases Mining Points from Quest Bounty.

Player impact:

* The same accepted quest can produce more Mining Points.
* Power is a reliable reward multiplier for active miners.
* Power grants from items improve the trait-derived bonus rather than replacing the need for Mining investment.

## Flow

Flow adds a Mining Points bonus when Flow is active.

Flow is a streak mechanic. After a successful mining action, the character can keep Flow active for a period controlled by Focus. If the next successful action happens inside that window, Flow helps the reward.

Player impact:

* Rewards active sessions and consistent quest completion.
* Makes timing matter.
* Works best together with Focus.

## Focus

Focus controls how long Flow remains active.

Player impact:

* More Focus gives more time between successful actions.
* It makes Flow easier to maintain for players who cannot complete actions back-to-back.
* The live window is capped at `12 hours`.

## Priority

Priority affects moderation queue order for completions that need moderation.

It does not skip moderation and does not guarantee approval. Waiting time still matters: older submissions continue to move forward. Priority makes the player's completion count as stronger inside the queue.

Player impact:

* Useful for players whose quests often require moderation.
* Helps reduce waiting pressure without removing the moderation layer.
* Priority grants can add direct queue advantage.

## Loot

Loot controls Chest Shard roll rate from successful quest completion.

Below `100%`, Loot is a chance for a shard roll. Above `100%`, it can create guaranteed shard rolls plus a chance for an extra one. The shard piece itself is then influenced by [Luck Shards](luck.md#shards).

Player impact:

* More shard rolls mean more chances to complete the weekly shard puzzle.
* Completing shard sets creates free Common Lootbox opportunities.
* Loot and Luck work together: Mining creates more rolls, Luck improves the chance that rolls help finish the set.
